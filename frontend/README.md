# Frontend - WebApp Académico

Frontend Vue 3 + TypeScript para la aplicación académica.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Estructura

- `src/components/` - Componentes Vue reutilizables
- `src/views/` - Vistas por rol (Estudiante, Docente, Admin)
- `src/stores/` - State management (Pinia - por implementar)
- `src/assets/` - Imágenes y estilos globales
