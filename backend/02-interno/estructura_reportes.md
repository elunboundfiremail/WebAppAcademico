# Estructura de reportes RF12

## 1. Reporte de materias por carrera (Administrador)
- Formato: PDF y Excel
- Campos requeridos:
  - Codigo de carrera
  - Nombre de carrera
  - Codigo de materia
  - Nombre de materia

## 2. Reporte de notas por estudiante (Docente)
- Formato: PDF y Excel
- Campos requeridos:
  - Nombre del estudiante
  - Correo del estudiante
  - Materia
  - Grupo
  - Nota de evaluacion
  - Tipo de evaluacion
  - Fecha de registro

## 3. Reporte de historial academico (Estudiante)
- Formato: PDF
- Campos requeridos:
  - Datos del estudiante (Nombre, Correo)
  - Lista de materias cursadas
  - Periodo academico
  - Calificacion final por materia
  - Estado (Aprobado/Reprobado)
